# PrintBeautifulLog

### 作用
将字典(NSDictionary)和数组(NSArray)打印的Log显示为Json格式,可以解决Xcode打印字典和数组时中文显示乱码问题。

### 使用方式
直接将 NSDictionary+Log分类和NSArray+Log分类文件拉入到工程中即可自动生效使用。
